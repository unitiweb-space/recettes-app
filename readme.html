PRÉSENTATION : Ce projet a pour but de créer un système de recettes de cuisine basé sur les CPT de WordPress et les champs personnalisés. Le thème utilisé est un thème enfant de Twenty Twenty Four, natif dans WordPress et basé sur les blocks. OUTILS UTILISÉS : Carbon fields : alternative open source de ACF qui permet aussi de créer des champs repéteurs. Recette Application : création d'un block sur mesure qui permet d'afficher la liste des recettes en page d'accueil. Recette Application single : création d'un block sur mesure qui permet d'afficher une recette en mode single. Thème enfant : thème enfant de Twenty Twenty Four créé pour l'occasion. DIFFICULTÉS RENCONTRÉES : - Trouver une alternative à ACF. - Se former rapidement à Carbon fields. - Adapter le système à un thème basé sur les blocks et utilisant des templates html. - Uniformiser le CSS sur toutes les pages. ÉTAPES DE CRÉATION DE L'OUTIL : 1. Installation/initialisation de Carbon fields. 2. Création de deux blocks pour l'affichage sur la home et l'affichage des détails de la recette sur un modèle single-recette.html. 3. Création/Paramétrage des champs personnalisés. 4. Création des recettes. INSTALLATION ET PARAMÉTRAGE: 1. Installer le plugin Carbon fields. 2. Copier les blocks "recette-application" et "recette-application-single" disponibles dans le dossier plugins. 3. Installer le thème enfant pour faire apparaitre les champs personnalisés dans l'admin.
